RestService

This is RestService. You can search for products by going to http://aalto-digi-restservice.herokuapp.com/products/ and adding url query parameters.

Parameters

limit - Give an upper limit of search results to return. Can give many, but only last parameter is processed. Only the last Number, negative values are interpreted as 0.

notfield - Give the name of a field that should not be included in the results, i.e "price" will return results that do not give their prices. String

category and notcategory - The category(ies) that should or shouldn't be included in the results. String

subcategory and notsubcategory - As with categories, only the more specific subcategory or genre. String

maxprice - Set the exclusive maximum price. I.e specifying 400 will return items whose price is at most 399.99. Number.

sort and dsort - Give the name of a field by which to sort up (sort) or down (dsort). Can give many. String

minamount - Give the inclusive minimum amount of product that should be in stock. I.e giving 0 means no products that are out of stock are returned. Can give many, but only last parameter is processed. Number
